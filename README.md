# בודק עבודות אקדמיות - תוסף לגוגל דוקס

תוסף ל-Google Docs שבודק הצעות מחקר לתואר ראשון. התוסף בודק מבנה, ביבליוגרפיה (APA7), ומספק ביקורת בונה על התוכן באמצעות AI.

## מה התוסף בודק?

1. **דף שער** - שם עבודה, שם סטודנט/ית, שם מנחה, תאריך
2. **מבנה פרקים** - מבוא (פרק 1), סקירת ספרות (פרק 2), מתודולוגיה (פרק 3), ביבליוגרפיה (פרק 4)
3. **ביבליוגרפיה APA7** - בדיקת פורמט, סדר אלפביתי, DOI, עדכניות מקורות
4. **סקירת תוכן** - ביקורת בונה לכל פרק (מחמאות + הצעות לשיפור), בדיקת איכות כתיבה ורמה אקדמית
5. **בדיקת פרק השיטה לעומק** - בהירות, שלמות, ניסוח, כלי מחקר, אוכלוסייה, שיקולים אתיים
6. **סיכום והחלטה** - אישור הצעת מחקר והכוונה להגשת מדריך ראיון

## התקנה

### אפשרות 1: התקנה ידנית (מומלץ)

1. פתח מסמך Google Docs
2. לחץ על **Extensions** (תוספים) > **Apps Script**
3. מחק את הקוד הקיים בקובץ `Code.gs`
4. צור את הקבצים הבאים והעתק לתוכם את הקוד המתאים:

| קובץ ב-Apps Script | קובץ מקור |
|---|---|
| `Code.gs` | `src/Code.gs` |
| `StructureChecker.gs` | `src/StructureChecker.gs` |
| `APAChecker.gs` | `src/APAChecker.gs` |
| `ContentReviewer.gs` | `src/ContentReviewer.gs` |
| `Sidebar.html` | `src/Sidebar.html` |

5. החלף את תוכן `appsscript.json` (לחץ על Project Settings > Show "appsscript.json" manifest file)
6. שמור את כל הקבצים (Ctrl+S)
7. חזור למסמך ורענן את הדף

### אפשרות 2: התקנה עם clasp

```bash
npm install -g @google/clasp
clasp login
clasp create --type docs --title "בודק עבודות אקדמיות"
```

העתק את כל הקבצים מ-`src/` לתיקיית הפרויקט ואז:

```bash
clasp push
clasp open
```

## הגדרת API (נדרש לסקירת תוכן)

הבדיקות המבניות (דף שער, פרקים, APA7) עובדות ללא API.
לסקירת תוכן מעמיקה עם AI, נדרש מפתח Gemini API:

1. היכנס ל-[Google AI Studio](https://aistudio.google.com/app/apikey)
2. צור מפתח API (חינמי)
3. בגוגל דוקס: **בודק עבודות** > **הגדרות API** > הכנס את המפתח

## שימוש

1. פתח את הצעת המחקר ב-Google Docs
2. לחץ על **בודק עבודות** > **הפעל בדיקה מלאה** בתפריט
3. ייפתח סרגל צד עם כפתור בדיקה
4. לחץ על **הפעל בדיקה מלאה**
5. תוצאות הבדיקה יוצגו בסרגל הצד, מחולקות לפי נושא

## מבנה הפרויקט

```
src/
  Code.gs              - נקודת כניסה, תפריט, הגדרות
  StructureChecker.gs  - בדיקת דף שער ומבנה פרקים
  APAChecker.gs        - בדיקת ביבליוגרפיה APA7
  ContentReviewer.gs   - סקירת תוכן עם Gemini AI
  Sidebar.html         - ממשק משתמש (עברית, RTL)
appsscript.json        - הגדרות manifest
```

## דרישות מהמסמך

כדי שהתוסף יעבוד באופן מיטבי, ודאו שהמסמך עומד בדרישות הבאות:

- **כותרות** - כל שם פרק מוגדר כ-Heading (כותרת) ב-Google Docs
- **דף שער** - מופיע בתחילת המסמך עם הפרטים הנדרשים
- **ביבליוגרפיה** - מתחילה עם כותרת "ביבליוגרפיה" / "רשימת מקורות" / "References"
- **רשומות** - כל רשומה ביבליוגרפית בשורה/פסקה נפרדת
